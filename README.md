## Technologies, packages used in this project:
- FrontEnd: Vite + hook useContext & React query 
- BackEnd: NodeJS, Express, Prisma for interactting with database(MongoDB) 
- Payment: Stripe

## To running develop environment:
- BackEnd:
  1. cd server/
  2. Install yarn 
  3. yarn run start
- FrontEnd:
  1. cd client/
  2. Install yarn 
  3. yarn run dev
